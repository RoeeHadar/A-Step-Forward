# Fraction–Decimal Links

Every fraction can be viewed as division: **3/4** means 3 ÷ 4, which equals **0.75**.

## Common conversions

| Fraction | Decimal |
| -------- | ------- |
| 1/2      | 0.5     |
| 1/4      | 0.25    |
| 3/4      | 0.75    |

## Try it

Convert **1/5** to a decimal. Show the division you used.
