# Adding Fractions with Like Denominators

When fractions have the **same denominator**, you add the numerators and keep the denominator:

**2/7 + 3/7 = 5/7**

Think of seventh-sized slices: 2 slices plus 3 slices make 5 slices, still each one-seventh of the whole.

## Why denominators must match

Denominators tell you the **size** of each part. You can only combine parts of the same size directly.

## Try it

Compute **1/8 + 4/8**. Then write a word problem that this equation could represent.
