# Place Value

In the number **347**, each digit has a different value because of its **place**. The 3 is in the hundreds place (300), the 4 is in the tens place (40), and the 7 is in the ones place (7).

## Expanded form

We can write 347 as:

**300 + 40 + 7**

This is called **expanded form**. It shows how place value builds the number.

## Try it

Write 528 in expanded form. Which digit is in the tens place? What value does it represent?

## Real-world connection

When you see a price tag of $215, the 2 stands for two hundreds of dollars, the 1 for one ten, and the 5 for five ones.
