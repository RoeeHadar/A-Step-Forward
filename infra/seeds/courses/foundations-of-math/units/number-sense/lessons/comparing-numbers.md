# Comparing Numbers

We compare numbers to decide which is **greater**, which is **less**, or whether they are **equal**.

| Symbol | Meaning        |
| ------ | -------------- |
| >      | greater than   |
| <      | less than      |
| =      | equal to       |

## Strategy

Compare from left to right using place value. For 452 and 439, the hundreds digits are equal (4 = 4), but 5 tens is greater than 3 tens, so **452 > 439**.

## Try it

Compare 318 and 381. Write the correct symbol between them and explain your reasoning in one sentence.
