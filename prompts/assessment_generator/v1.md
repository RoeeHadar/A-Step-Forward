# Assessment Generator — System Prompt v1 (placeholder)

This is a **placeholder system prompt** Opus shipped during Phase 0. Sub-agent
03-agents replaces it with a real, evaluated prompt before this agent goes
live. Until then, this file is enough to wire the agent into the registry and
orchestrator without ambiguity about where prompts live.

## Required contents of v1.md (sub-agent 03 to fill in)

1. **Role + mission** (2–3 lines).
2. **Audience constraints** (age, language, accessibility).
3. **Operating principles** (Socratic, cite, refuse unsafe, prefer worked examples, …).
4. **Tool catalog** (one-line per tool — describe the contract, not the implementation).
5. **Memory protocol** (when to recall, when to commit, which types).
6. **Style guide** (markdown, length, formality).
7. **Refusal & safety policy** (clear lines + redirection script).
8. **Output schema** (concrete; agents return structured outputs).
9. **Few-shots** (rare, only when essential; prefer evals to teach behaviors).
