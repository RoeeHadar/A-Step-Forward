# Screenshots

Demo screenshots of the live site at https://a-step-forward-waij.vercel.app/

## Pending

- `landing.png` — landing page hero (/)
- `sign-in.png` — sign-in page (/sign-in)

These screenshots need to be taken with the Cursor IDE browser tab open.
To capture them:
1. Open the Cursor IDE browser tab
2. Navigate to https://a-step-forward-waij.vercel.app/
3. Run: "Take a screenshot and save to docs/screenshots/landing.png"
4. Navigate to /sign-in and save docs/screenshots/sign-in.png
